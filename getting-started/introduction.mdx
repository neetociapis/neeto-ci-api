The NeetoCI API is organized around REST principles and uses standard HTTP response codes. All API endpoints return JSON responses and require authentication.

Before you start with NeetoCI APIs you will need the following:

1. [Workspace subdomain](/getting-started/workspace-subdomain).
2. [API key](/getting-started/authentication).

## Making your first API call

Let's create a CI job in your workspace using a POST request:

<CodeGroup>
```bash curl
curl -X POST "https://{your-subdomain}.neetoci.com/api/v2/ci-jobs" \
  -H 'X-Api-Key: your-api-key' \
  -H 'Content-Type: application/json' \
  -d '{
    "project_name": "your-project-name",
    "branch_name": "main",
    "workflow_name": "playwright-staging"
  }'
```

```javascript JavaScript
const response = await fetch(
  "https://{your-subdomain}.neetoci.com/api/v2/ci-jobs",
  {
    method: "POST",
    headers: {
      "X-Api-Key": "your-api-key",
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      project_name: "your-project-name",
      branch_name: "main",
      workflow_name: "playwright-staging"
    })
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests
import json

url = "https://{your-subdomain}.neetoci.com/api/v2/ci-jobs"
headers = {
    "X-Api-Key": "your-api-key",
    "Content-Type": "application/json"
}
payload = {
    "project_name": "your-project-name",
    "branch_name": "main",
    "workflow_name": "playwright-staging"
}

response = requests.post(url, headers=headers, json=payload)
data = response.json()
print(data)
```

```php PHP
<?php
$curl = curl_init();

$payload = json_encode([
    "project_name" => "your-project-name",
    "branch_name" => "main",
    "workflow_name" => "playwright-staging"
]);

curl_setopt_array($curl, [
    CURLOPT_URL => "https://{your-subdomain}.neetoci.com/api/v2/ci-jobs",
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_POSTFIELDS => $payload,
    CURLOPT_HTTPHEADER => [
        "X-Api-Key: your-api-key",
        "Content-Type: application/json"
    ],
]);

$response = curl_exec($curl);
curl_close($curl);

$data = json_decode($response, true);
print_r($data);
?>
```

</CodeGroup>

<Warning>
  Replace `your-subdomain` with your actual workspace subdomain,
  `your-api-key` with your actual API key, and
  `your-project-name` with the name of your project in NeetoCI.
</Warning>

## Rate Limiting

API requests are subject to rate limiting to ensure fair use. If you exceed the rate limit, you'll receive a 429 response. If you encounter this error try again after some time.
