## Overview

This endpoint allows you to trigger a CI job for a specific project and branch with a specified workflow.

## Request Body

The request body should contain the following fields:

- `branch_name` (required): The name of the branch to trigger the CI job for
- `project_name` (required): The name of the project to trigger the CI job for
- `workflow_name` (required): The name of the workflow to execute

## Example Request

```bash
curl -X POST https://neetozone.neetoci.com/api/v1/trigger_ci_job \
  -H "X-Api-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "branch_name": "main",
    "project_name": "neeto-cal-web",
    "workflow_name": "playwright-nightly-staging"
  }'
```

## Example Response

```json
{
  "message": "CI job triggered successfully",
  "job_id": "ci-job-12345",
  "status": "queued",
  "project": "neeto-cal-web",
  "branch": "main",
  "workflow": "playwright-nightly-staging",
  "triggered_at": "2024-01-15T10:30:00Z"
}
```
